# UML - Modelagem Básica Iphone
Treinar modelagem de software, proposta da DIO 
## Do que se trata
Desafio proposto para criar um diagrama representativo de um Iphone (2007), a fim de treinar a abstração e a relação entre as classes de um sistema.

> Nota: Aqui encontra-se somente o esquema UML, criado com o [DRAW.IO](https://www.drawio.com/).
